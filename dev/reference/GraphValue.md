# Graph Value

Value in an
[`AnvilGraph`](https://r-xla.github.io/anvil/dev/reference/AnvilGraph.md).
This is a mutable class.

## Usage

``` r
GraphValue(aval)
```

## Arguments

- aval:

  ([`AbstractTensor`](https://r-xla.github.io/anvil/dev/reference/AbstractTensor.md))  
  The abstract value of the variable.
