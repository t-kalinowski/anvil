# Primitive Concatenate

Concatenates tensors along a dimension.

## Usage

``` r
nvl_concatenate(..., dimension)
```

## Arguments

- ...:

  ([`tensorish`](tensorish.md))  
  Tensors to concatenate.

- dimension:

  (`integer(1)`)  
  Dimension to concatenate along.

## Value

[`tensorish`](tensorish.md)
