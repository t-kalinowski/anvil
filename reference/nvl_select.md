# Primitive Select

Selects elements based on a predicate.

## Usage

``` r
nvl_select(pred, true_value, false_value)
```

## Arguments

- pred:

  ([`tensorish`](tensorish.md))  
  Boolean predicate tensor.

- true_value:

  ([`tensorish`](tensorish.md))  
  Value when pred is true.

- false_value:

  ([`tensorish`](tensorish.md))  
  Value when pred is false.

## Value

[`tensorish`](tensorish.md)
