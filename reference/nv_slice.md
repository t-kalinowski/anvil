# Slice

return slice of operand.

## Usage

``` r
nv_slice(operand, start_indices, limit_indices, strides)
```

## Arguments

- operand:

  ([`nv_tensor`](nv_tensor.md))  
  Operand.

- start_indices:

  start of slice

- limit_indices:

  end of slice

- strides:

  stride size

## Value

[`nv_tensor`](nv_tensor.md)
