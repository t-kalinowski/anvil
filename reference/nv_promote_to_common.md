# Promote Tensors to a Common Dtype

Promote tensors to a common data type, see
[`common_dtype`](common_dtype.md) for more details.

## Usage

``` r
nv_promote_to_common(...)
```

## Arguments

- ...:

  ([`tensorish`](tensorish.md))  
  Tensors to promote.

## Value

([`list()`](https://rdrr.io/r/base/list.html) of
[`tensorish`](tensorish.md))
